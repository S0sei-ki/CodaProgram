Pygame Godot Scratch

Godot is an open-source game engine that offers a flexible and intuitive platform for both 2D and 3D game development. It features a visual editor, a powerful scripting language (GDScript), and a node-based architecture that simplifies game design. With cross-platform support and a vibrant community, Godot is ideal for indie developers and hobbyists looking to create polished games quickly and efficiently.
